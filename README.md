🌍 Global Trade Dashboard (Power BI)

An interactive Power BI dashboard analyzing 30+ years of global export-import data to identify trade patterns, regional trends, and country-level performance.

## 📌 Overview
This dashboard provides a comprehensive view of international trade across 200+ countries from 1988 to 2022. It helps identify top trading partners, trade balance trends, and duty structures across different years.

## 🚀 Key Insights
- Total Trade Volume analyzed: **688.73 Billion USD**
- Top trading partners identified by export and import volume
- Country-wise trade balance and % growth tracked year over year
- Duty-free vs dutiable imports analyzed across 30+ years

## 🛠️ Tech Stack
- **Tool:** Power BI Desktop
- **Data:** World Bank Global Trade Dataset (Import/Export)
- **Features:** Dynamic filters, KPIs, World Map visuals, Donut charts, Line trends

## 📊 Dashboard Features
- Interactive world map showing trade volumes by country
- Year and country filters for drill-down analysis
- Export vs Import comparison by partner name
- AHS and MFN duty structure analysis over time
- Trade balance and growth % KPIs
